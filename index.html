Building UI for some Exercism JavaScript challenges.

A beginner project to test my frontend skills.

01

Allergies

Given a person's allergy score, determine whether or not they're allergic to a given item, and their full list of allergies.

02

Scrabble Score

Given a word, compute the Scrabble score for that word.

03

Resistor Color Duo

The program will take two color names as input and output a two digit number, even if the input is more than two colors!

04

Random Numbers Generator

Enter the number of randomly generated numbers you want

05

Space Age

Select a planet and enter an age in seconds to get the actual age on that planet.

06

Lucky Number

Given a number, determine whether the number is a palindrone.

07

Word Count

Given a sentence, return the number of occurrences of each word
